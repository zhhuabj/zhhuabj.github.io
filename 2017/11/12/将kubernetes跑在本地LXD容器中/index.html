LXD上安装Kubernetes

1, 从这个链接下载 ‘canonical-kubernetes.zip’ ，里面有下面要用到的bundle.yaml
2, 使用 juju-default profile运行’juju bootstrap’
juju bootstrap –debug –config bootstrap-series=xenial –config agent-stream=devel localhost juju-default
3, 使用juju一键部署kubernetes
juju deploy bundle.yaml
4，部署完毕
